# Automobile Industry Sales Performance Analysis India (2024)

## Project Overview
This project analyses 2024 automobile industry sales data using Power BI.  
The dashboard provides insights into total sales, market share, brand performance, and time-based trends.

## Tools Used
- Power BI
- DAX
- Data Modeling
- Power Query

## Features
- KPI cards for Total Sales and Market Share
- Brand-wise performance analysis
- Drill-through report pages
- Time intelligence using Date table
- Interactive filters and slicers

## Skills Demonstrated
- Data Analysis
- Business Intelligence
- Dashboard Development
- Data Modeling
- KPI Reporting

## Files Included
- Power BI Dashboard
- Dataset
- Dashboard screenshots

## Purpose
This project demonstrates data analysis, dashboard design, and business intelligence skills for Data Analyst role.
